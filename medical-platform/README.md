# 🏥 المنصة الطبية المتكاملة

منصة حجز طبي متكاملة مع خرائط تفاعلية للحجز والاستشارات الطبية.

## ✨ المميزات

- 🗺️ **خريطة تفاعلية**: اعثر على أقرب العيادات، المستشفيات، والصيدليات
- 👨‍⚕️ **أطباء متخصصون**: احجز مع أفضل الأطباء حسب الاختصاص
- 💊 **صرف إلكتروني**: احصل على وصفاتك الطبية إلكترونياً
- 📅 **حجز مواعيد**: احجز موعدك في أي وقت وبسهولة
- 🔐 **خصوصية كاملة**: كل موقع لا يعلم عن الآخر
- 👑 **مستخدم خارق (Super User)**: لوحة تحكم شاملة لإدارة كل شيء

## 🚀 النشر على Render.com

### الخطوات:

1. **ارفع المشروع على GitHub**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin YOUR_GITHUB_REPO
   git push -u origin main
   ```

2. **أنشئ قاعدة بيانات PostgreSQL على Render**
   - اذهب إلى https://render.com
   - أنشئ قاعدة بيانات PostgreSQL جديدة
   - انسخ رابط الاتصال (DATABASE_URL)

3. **أنشئ خدمة ويب جديدة على Render**
   - اختر "Web Service"
   - اربط مستودع GitHub الخاص بك
   - استخدم الإعدادات التالية:
     - **Build Command**: `npm install && npm run build`
     - **Start Command**: `npm start`
     - **Environment**: Node

4. **أضف متغيرات البيئة**
   - `DATABASE_URL`: رابط قاعدة البيانات من الخطوة 2
   - `NEXTAUTH_SECRET`: مفتاح سري عشوائي
   - `NEXTAUTH_URL`: رابط خدمتك على Render

5. **تهيئة قاعدة البيانات**
   ```bash
   npx prisma migrate deploy
   npx prisma generate
   ```
