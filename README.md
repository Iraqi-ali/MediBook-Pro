# 🏥 MediBook Pro - نظام الحجز الطبي المتكامل

![MediBook Pro](https://img.shields.io/badge/version-1.0.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)
![Node](https://img.shields.io/badge/node-%3E%3D14.0.0-brightgreen.svg)

نظام حجز طبي متكامل مع خرائط تفاعلية وإدارة كاملة للعيادات والصيدليات

## ✨ المميزات

- 🗺️ **خرائط تفاعلية** مع أيقونات مخصصة للعيادات والصيدليات
- 📅 **حجز المواعيد** أونلاين مع إمكانية الاختيار حسب التخصص والموقع
- 💊 **صرف الأدوية إلكترونياً** من العيادات والصيدليات
- 🎥 **استشارات عن بعد** عبر الفيديو
- 📱 **متجاوب مع جميع الأجهزة** (حاسوب، هاتف، جهاز لوحي)
- 🏪 **3 أشكال للصفحات** (عيادة، صيدلية، مجمع طبي)
- 🔐 **نظام صلاحيات متعدد** (سوبر يوزر، مدير، طبيب، مريض)
- 📊 **لوحة تحكم احترافية** لكل نوع مستخدم
- 🌐 **دعم كامل للغة العربية** RTL
- 🎨 **واجهة مستخدم عصرية وجذابة**

## 🚀 التقنيات المستخدمة

### الواجهة الأمامية (Frontend)
- React.js 18
- Material-UI (MUI)
- Leaflet Maps
- Redux Toolkit
- Socket.io Client
- Framer Motion

### الخلفية (Backend)
- Node.js & Express
- MongoDB & Mongoose
- JWT Authentication
- Socket.io
- Redis
- Cloudinary

## 📋 المتطلبات

- Node.js (v14 أو أحدث)
- MongoDB (v4.4 أو أحدث)
- Redis (اختياري - للتخزين المؤقت)
- npm أو yarn
